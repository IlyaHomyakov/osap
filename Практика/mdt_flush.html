Параллельные связи между двумя плоскостями

Опция Flush используется для размещения поверхностей в одной плоскости (заставляет поверхности оставаться в одной плоскости).

Команда AMFLUSH используется для создания параллельных связей в указанном смещении между двумя плоскостями. Поскольку связи узла являются параметрическими, можно изменять их параметры в любое время. Способы обращения к команде AMFLUSH:

  • щелкните правой кнопкой мыши в графической области и разверните на экране контекстно-зависимое меню. Щелчком по пункту 3D Constraints (Трехмерные связи) вызовите всплывающее меню. Щелкните по пункту Flush (Сброс);
  • щелкните по кнопке Flush на панели инструментов;
  • щелкните по пункту Assembly (Узел) главного меню, по пункту 3D Constraints выпадающего меню, а затем по пункту Flush всплывающего меню;
  • введите в командной строке команду AMFLUSH.

В результате увидите запрос:

Select first set of geometry:
(Укажите первый набор геометрии)

Дальнейшие действия:

1. Выберите первую плоскость. Последуют сообщение и запрос:

First set = Plane
(Первый набор = Плоскость)
Enter an option [Clear/Next/Flip/cYcle]<accept>:
(Введите опцию [Очистить/Следующая/Отразить зеркально/Цикл]<принять>)

2. Введите опцию или нажмите клавишу Enter. Появится запрос:

Select second set of geometry:
(Укажите второй набор геометрии)

3. Отметьте вторую плоскость. Поступят сообщение и запрос:

Second set = Plane
Enter an option [Clear/Next/Flip/cYcle]<accept>:

4. Задайте опцию или нажмите на Enter. Поступит предложение:

Enter offset <0.0000>:
(Введите смещение <0.0000>)

5. Введите смещение или нажмите клавишу Enter. В запросе присутствует несколько опций:

  • Clear (Очистить) - очищает текущий выбор;
  • Next (Следующая) — переключает на следующую доступную геометрию: ось, грань или точку; Flip (Отразить зеркально) - зеркально отражает направление нормали;
  • Cycle (Цикл) - циклически проходит доступную геометрию.

В процессе работы с командой появляется анимационный курсор, указывающий направление вращения. Если он снабжен анимационными стрелками, воспользуйтесь левой кнопкой мыши, чтобы циклически пройти доступные опции, и правой - чтобы завершить выбор.
