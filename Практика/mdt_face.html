Пример рассечения грани


Пример рассечения грани на две детали

Проецируемое рассечение грани позволяет создать объёмные тела рассекая одну из граней тела эскизом линии разреза. Порядок создания рассечения грани следующий:

  • Имеется параллелепипед;
  • На его боковой грани создаётся эскиз секущей линии, для чего UCS выравнивается по этой грани;
  • Плоскость эскиза синхронизируется с UCS (New Sketch Plane);
  • Эскиз секущей линии преобразуется в Split Line (Profile a sketch > Split Line);
  • Деталь рассекается на две детали командой ampartsplit (Placed Features > Part Split). После этого MDT запросит имя для второй детали.
