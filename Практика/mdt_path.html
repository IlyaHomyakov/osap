Эскизы траекторий могут быть двух (2D) и трёхмерными (3D). В отличие от эскизов профиля, эскизы тракторий обычно разомкнуты.


Эскиз 2D траектории

Эскиз 2D траектории является путём для перемещаемого (протягиваемого) вдоль него профиля поперечного сечения. Определяемое таким образом пространство является телом протягивания. Это может быть изогнутая труба, в случае использования 2D траектории для создания нового тела, либо фигурная канавка на поверхности детали, в случае задания 2D траекторией пути перемещаемого резца, который извлекает материал.

При создании 2D траектории плоскость эскиза должна совпадать с плоскостью используемых для её построения примитивов. Эскиз 2D траектории может включать линии, дуги, полилинии, сегменты эллипса и 2D сплайны. При определении эскиза 2D траектории, можно автоматически создавать рабочую плоскость в начальной точке (Work Point) перпендикулярную траектории протягиваемого профиля, а также привязки эскиза профиля к начальной точке траектории.

При запросе начальной точки траектории укажите точку, где начинается траектория. Это определит направление при протягивании профиля поперечного сечения.

Эскиз 3D траектории спирали

Траектория трехмерной спирали используется при создании элемента протягивания (sweep). Винтовое протягивание используется для создания резьб, пружин и катушек. При этом создаётся траектория трёхмерной спирали по существующей рабочей оси, цилиндрической поверхности или цилиндрической кромке.

При создании траектории трёхмерной спирали, можно указать, будет ли создаваться рабочая плоскость. Она может быть перпендикулярна траектории в её центре или по рабочей оси. Эта рабочая плоскость будет использоваться при отрисовке эскиза профиля для протягивания по спирали.

Для создания траектории трехмерной спирали необходимо проделать следующие шаги:

Шаг 1

Создать рабочую ось.

Шаг 2

С помощью команды UCS повернуть систему координат так, чтобы она была перпендикулярна созданной рабочей оси.

Шаг 3

Задать плоскость эскиза (New Sketch Plan), лежащую в текущей системе координат, для чего:

Браузер

Нажать правую клавишу мыши на поле Браузера и выбрать команду New Sketch Plan.

(Work Axis)

      

Меню

Диалоговое окно Helix

TYPE

Определяет тип создаваемой траектории.

Pitch and Revolution. Создает трехмерную траекторию, используя указанное количество витков и шаг между витками.

Revolution and Height. Создает трехмерную траекторию с указанными количеством витков и полной высотой.

Height and Pitch. Создает трехмерную траекторию, используя полную высоту и шаг между витками.

Spiral. Создает плоскую траекторию по количеству витков и шагу между витками.

ORIENTATION

Управляет направлением и углом начала траектории. Указывает, поворачивается ли траектория по часовой стрелке или против и определяет начальный угол от плоскости эскиза.

SHAPE

Управляет формой траектории, её полным диаметром, первой и второй осями эллиптической траектории и углом наклона.

Circle. Создает круговую траекторию.

Ellipse. Создает эллиптическую траекторию.

Для круговой траектории необходимо указать диаметр, а для эллиптической, - первую и вторую оси.

Taper Angle. Задаёт угол наклона траектории.

PROFILE PLANE

Управляет созданием рабочей плоскости и её расположением.

None. Рабочая плоскость не создается.

Center Axis/Path. Рабочая плоскость совмещается с началом траектории, а для траектории используется рабочая ось.

Normal To Path. Рабочая плоскость создаётся перпендикулярно к точке начала траектории.
