Изменение текущего стиля линии

Текущий стиль линии определяет толщину, тип, цвет линий которыми будет отрисован следующий объект. Значения параметров линий по умолчанию задаются в описании слоя, однако можно задавать свойства линий, отличные от установок, заданных в текущем слое. Наиболее простой способ выбора параметров линии - выбирать их из выпадающих списков панели инструментов, находящейся прямо над рабочей областью чертежа.

Первый выпадающий список касается текущего слоя, в нем даны названия слоев и значки, показывающие их основные параметры.

Второй список имеет уже непосредственное отношение к линиям: из него можно задать цвет линии. Кроме непосредственного указания цветов, там присутствует пункт By layer, устанавливающий цвет линии по слою. Если цветов, представленных в списке по умолчанию, не хватает, можно выбрать дополнительные с помощью пункта Other выпадающего списка. На экране появляется диалоговое окно Select color, из которого можно выбрать нужный цвет.

Выбранный цвет появляется в списке.

Следующий список содержит типы линий. В нем также имеется пункт By layer - установка типа линии по слою.

По умолчанию подавляющее большинство типов линий не загружено в данный список и соответственно не отображается в выпадающем списке типов линий. Для того чтобы загрузить дополнительные типы линий, необходимо использовать кнопку Load. В результате на экране появляется окно выбора типов линий для загрузки Load or Reload linetypes. Выбирая в нем тип линий и нажимая ОК, можно сделать его доступным для использования.

Для установки толщины линий используется четвертый по счету выпадающий список, в котором отображаются возможные величины толщины линий и их примеры.
