Командная строка

ELLIPSE

Команда ELLIPSE, в зависимости от значения системной переменной PELLIPSE, строит эллипс как замкнутую полилинию, состоящую из коротких дуговых сегментов, или как эллипс. Полилиния строится при значении 1, эллипс - 0.

При наборе команды с клавиатуры по умолчанию она работает как пункт меню Axis, End и выдает сначала запрос:

Arc/Center/<Axis endpoint 1>:
Дуга/Центр/<Точка оси 1>:

В ответ на него нужно указать первую точку оси эллипса либо ввести ключ С, после чего следует указывать центр эллипса (так работает пункт меню Center). Потом появляется запрос:

Axis endpoint 2:
Точка оси 2:

В ответ на него следует указать другой конец оси эллипса. Далее появляется запрос:

<Other axis distance>/Rotation:
<Величина другой оси>/Поворот:

В ответ на него следует ввести половину длины другой оси либо ключ Rotation.

При использовании ключа Rotation (поворот) первая указанная ось приобретает статус большой. Под большой осью понимается диаметр воображаемой окружности, вокруг которого она поворачивается в пространстве, проецируясь на неподвижную плоскость в виде эллипса. Величину угла можно задать, введя численное значение угла в градусах от 0 до 89,4. При задании угла путем указания точки углом поворота круга считается угол между большой осью и лучом, проведенным из ее средины в указанную точку.

Для построения эллиптических дуг используется ключ Arc, вводимый в ответ на первый запрос.

Дуги строятся подобно тому, как это делалось при построении эллипса, но задаются дополнительные значения:

  • Угла - ключ Angle, с помощью которого определяются начальный (start angle) и конечный угол (end angle), которые измеряются от линии, проходящей через центр эллипса в направлении главной оси. Начальный угол определяет начальную точку эллиптической дуги, а конечный - конечную точку. При этом рисуется дуга в направлении, определяемом значением системной переменной ANGDIR. Если значение переменной 0, то дуга строится против часовой стрелки, если 1, то - по часовой стрелке.
  • Включающего угла - ключ Included (внутренний), с помощью которого определяются начальный и включающий дугу (Included angle (внутренний угол)) углы.
