Создание и использование блоков

Формирование объектов, которые часто используются, может быть произведено один раз. Затем они объединяются в блок, и чертеж может выполняться с использованием их как "строительных материалов". Применяя блоки, можно создать фрагменты чертежей, часто используемых в работе. Блок может содержать любое количество графических примитивов любого типа, а воспринимается AutoCAD как один графический примитив наравне с отрезком, окружностью и т.д.

Блок может состоять из примитивов, созданных на разных слоях, с разными цветами и разными типами линий. Все эти свойства примитивов сохраняются при объединении их в блок и при вставке блока в рисунок. Однако есть три исключения из этого правила:

  • примитивы, созданные на специальном слое с именем 0, при вставке блока генерируются на текущем слое;
  • примитивы, созданные в цвете BYBLOCK, наследуют цвет блока;
  • примитивы, созданные типом линии BYBLOCK, наследуют тип линии блока.

Блок может включать в себя другие блоки. Если внутренний блок включает примитивы, созданные на слое 0, или примитивы с цветом и типом линии BYBLOCK, то эти примитивы "всплывают" вверх сквозь вложенную структуру блоков до тех пор, пока не попадут в блок с "фиксированным" слоем, цветом или типом линии, иначе они генерируются на слое 0.

Блоку может быть присвоено имя. AutoCAD создает блоки и без имени (анонимные блоки), например, для ассоциативных размеров, т.е. для примитивов, к которым нет прямого доступа пользователя.

Использование блоков позволяет значительно сэкономить память. При каждой новой вставке блока в рисунок AutoCAD добавит к имеющейся информации лишь данные о месте вставки этого блока, масштабных коэффициентах и угле поворота.

С каждым блоком можно связать атрибуты, т.е. текстовую информацию, которую пользователь может изменять в процессе вставки блока в рисунок и которая может изображаться на экране или оставаться невидимой.

Команда ВМАКЕ - определение блока с помощью диалогового окна Block Definition (создание описания блока) и без удаления объектов из чертежа.

Команда BLOCK -создание описания блока из существующих объектов.


Создание блока

Кнопка инструментов
Make Block
     

Меню

Home | Draw | Block | Make

Командная строка

BMAKE

Для создания блока необходимо:

В строке Block name ввести имя блока, по которому к нему будут обращаться в дальнейшем.

В группе Base point задать базовую точку блока. Это можно сделать либо непосредственно вводом координат в строках X, Y, Z, либо, нажав кнопку Pick point и указав точку на чертеже в ответ на запрос

Insertion base point:

Далее следует выбрать объекты, которые будут входить в блок. Для этого следует нажать кнопку Select objects, после чего в ответ на запрос

Select objects:

указать мышью необходимые объекты.

В той же группе следует выбрать, что произойдет с выбранными объектами при включении их в блок. Если выбран пункт Retain Objects, объекты останутся в чертеже в первоначальном виде.

Команды BLOCK и ВМАКЕ создают блок, доступный для использования только в текущем чертеже. Для получения блоков, которыми можно воспользоваться при создании любых чертежей, в AutoCAD используется команда WBLOCK. Команда WBLOCK служит для записи блока в отдельный файл с заданным именем.


Вставка блока в чертеж

Предварительно определенные блоки или существующие файлы рисунков можно вставить в качестве блока в текущий чертеж с помощью команд INSERT и DDINSERT.

Кнопка инструментов
Insert Block
     

Командная строка

DDINSERT

Команда DDINSERT выводит диалоговое окно вставки блока Insert.

Для вставки блока в чертеж необходимо:

Набрать имя блока в строке Name или нажать на кнопку Block и выбрать имя. Если блок представляет собой файл на диске, то следует выбрать файл с помощью стандартного диалогового окна открытия файла, вызываемого кнопкой File.

Далее следует выбрать, каким образом будет указываться точка вставки. Если выбран пункт Specify Parameters On Screen в группе Options, то точка вставки будет указана мышью после закрытия диалогового окна. В противном случае точку вставки следует задать, введя значения координат в строки X, Y, Z в группе Insertion Point.

После этого следует задать масштабирование блока при вставке. Если выбран пункт Specify Parameters On Screen, масштаб будет задаваться при вставке блока с помощью мыши. В противном случае масштаб следует задать, введя значения в строки X, Y, Z в группе Scale.

Если необходимо при вставке повернуть блок, следует указать величину угла поворота в строке Angle группы Rotation либо выбрать пункт Specify Parameters On Screen для указания поворота с помощью мыши при вставке блока.

Если выбрать пункт Explode, блок будет вставлен не в виде единого объекта, а расчлененным на составные части.

После определения всех пунктов и нажатия кнопки ОК, если не были установлены координаты, появляется приглашение вставки блока:

Insertion point: X scale factor <1> / Corner / XYZ:
Точка вставки: Масштаб по оси X <1> / Угол / XYZ:

В ответ на него следует мышью указать, куда вставить блок.

Далее, если не были установлены масштабные коэффициенты, появляется приглашение:

Y scale factor (default=X):
Масштаб по оси Y <по умолчанию=X>:

В ответ на него следует указать коэффициент по оси Y либо нажать ENTER, чтобы принять тот же коэффициент, что и для X.

Далее следует указать угол поворота блока при вставке:

Rotation angle <0>:
Угол поворота <0>:

При включении блока в чертеж AutoCAD обрабатывает его как графический примитив. Для обеспечения работы с отдельными составляющими блок необходимо разбить. Для этого в AutoCAD существует команда EXPLODE - разбить блок на части, "рассыпать".

Если текущий рисунок является фрагментом, который будет помещаться в другие рисунки, для него можно задать базовую точку с помощью кoмaнды BASE.

Запрос:

Base point <0.0000,0.0000,0.0000>:

Если команда BASE не используется, то точка (0,0,0) становится по умолчанию его базовой точкой вставки. Кроме того, базовую точку вставки можно изменить, используя системную переменную INSBASE.

Можно создать множество вхождений блока, размещая их в вершинах прямоугольной сетки (массива), с помощью команды MINSERT - многократная вставка. При этом к рисунку добавляется только одно описание блока. В дополнение к запросам команды INSERT она выдает следующие запросы:

Number of rows (---) <1>:
Число строк (---) <1>:
Number of columns (|||) <1>:
Число столбцов (|||) <1>:
Unit cell or distance between rows (---):
Размер ячейки или расстояние между строками (---):
Distance between columns (|||):
Расстояние между столбцами (|||):

Угол поворота задает угол поворота всего массива вокруг точки вставки первого блока.
