Точка (Point)

Точка - самый простой геометрический объект. Геометрическая точка характеризуется тремя координатами. На экране точка может отображаться различными знаками (их около 20). Будьте осторожны - в наборе этих знаков есть знак под номером 1, который никак не отображает точку. Он введен для отключения точек из рассмотрения без их стирания.

Не следует путать точку с маркером: точка - примитив, элемент чертежа; к точке можно привязаться как к объекту; с ней можно производить многие операции редактирования; точки играют вспомогательную роль при построении чертежа. В AutoCAD точка используется, например, командами DIVIDE(ПОДЕЛИ) и MEASURE(РАЗМЕТЬ).

Можно использовать точки как вспомогательное средство для сложных геометрических построений. Удобно размещать их на отдельном слое и назначать отличный от других объектов цвет.
Закончив работу с точками как со вспомогательными примитивами, необязательно стирать все точки: можно просто отключить соответствующий слой.


Создание точки

Кнопка инструментов
Point
     

Меню

Home | Draw | Point

Командная строка

POINT

При исполнении данной команды выдается запрос:

Point:
Точка:

В ответ на него следует указать местоположение точки указанием ее координат. Координаты вводятся с клавиатуры (два или три числа, разделенных запятой; вместо десятичной запятой используйте точку) или непосредственно указывается точка в графической зоне экрана нажатием на левую клавишу мыши. После указания точки, в случае, если использовалась вторая команда, доступная из меню (кратная точка), будет выдан следующий аналогичный запрос, иначе выполнение команды прекратится. Способ отображения точки (видимый маркер) может быть задан с помощью системной переменной PDMODE, которая принимает следующие значения:

О - Точка (символ по умолчанию).
1 - Пусто.
2 - Прямой крест.
3 - Косой крест.
4 - Вертикальный Штрих от точки.

К каждому из этих значений можно добавить значения 32, 64 или 96. Toгда символ точки будет обрамлен некоторой фигурой:

32 - Круг.
64 - Квадрат.
96 - Круг и квадрат для значений PDMODE, отличных от 0 или 1.

Изменение размера маркера задается с помощью переменной PDSIZE.

Изменить вид точки и ее размер можно также с помощью диалогового окна, вызываемого с помощью меню Format -> Point Style.

Изменения этих переменных не влияют на ранее отрисованные точки: новые точки будут отрисовываться по-новому, а старые изменят "свой облик" только после регенерации рисунка.

Значения PDMODE и PDSIZE можно установить с помощью команды SETVAR или в диалоговом окне Point Style, которое вызывается с помощью пункта Point Style (отображение точек...) выпадающего меню Format или команды DDPTYPE.

При использовании команды SETVAR необходимо отвечать на запросы следующим образом:

Command:point
Команда:точка

Point:'setvar
Точка:'установить переменную

>>Variable name or ?:
>>Имя переменной или ?:

В ответ на этот запрос необходимо ввести изменяемую переменную PDMODE или PDSIZE. Изменим PDMODE.

>>New value for PDMODE <0>:34
>>Новое значение PDMODE <0>:34

Resuming POINT command.
Point:
Возобновляю команду точка.
Точка:
