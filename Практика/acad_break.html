Командная строка

BREAK

Данная команда вырезает из объекта часть, находящуюся между двумя вводимыми точками. Для того чтобы объект просто был разрезан на две половины без удаления его части, следует указать оба раза одну и ту же точку, которая и станет местом разрыва. При выполнении команды BREAK появляется следующий запрос:

Select objects:
Выберите объекты:

В ответ следует указать объект, который будет разрываться, щелкнув мышью на любой его точке. Далее появится запрос:

Enter second point (or F for first point):
Введите вторую точку (или П для первой точки):

После него пользователь должен ввести вторую точку разрыва либо букву F. В первом случае первой считается точка, указанная на объекте во время выбора его пользователем. Во втором- появятся запросы:

Enter first point:
Введите первую точку:
Enter second point:
Введите вторую точку:

После указания второй точки разрыва часть объекта между ними будет удалена.

Следует отметить, что при разрывании круга из него вырезается дуга от первой до второй точки против часовой стрелки.
