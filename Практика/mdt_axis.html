Рабочая ось

    Построение рабочей оси по цилиндру, конусу либо тору
    Построение рабочей оси по трём (двум) взаимоперпендикулярным рабочим плоскостям
    Построение рабочей оси по двум точкам существующего отрезка

Не все опорные элементы являются плоскостями. Некоторые лучше представлять в виде осей. Например, круглый цилиндр или сфера не имеют поверхностей, которые могли бы быть заданы в качестве плоских областей. Подобные объекты удобнее задавать рабочей осью. После задания рабочей оси, она может использоваться в качестве опорного элемента для рабочей плоскости или рабочей точки.

Рабочая ось может быть задана несколькими методами. Первый предполагает построение оси на основе существующего цилиндра, конуса либо тора. При этом можно указать какой-либо цилиндрический край. Второй метод основан на определении оси по двум точкам существующего отрезка. Рабочая ось также может быть задана и двумя взаимоперпендикулярными рабочими плоскостями.


Построение рабочей оси по цилиндру, конусу либо тору

Предположим, имеется цилиндрическая деталь

Шаг 1

Щёлкнем правой кнопкой мышки по графическому полю.
На экране раскроется контекстно-зависимое меню.

Шаг 2

Щелчком по пункту Sketched&Work Features (Эскизные и рабочие элементы) в контекстно-зависимом меню вызовем всплывающее меню.

Шаг 3

Щёлкнув в нём по пункту Work Axis (Рабочая ось), увидим следующее сообщение:

Select cylinder, cone, torus or [Sketch]: (Выберите цилиндр, конус, тор или [эскиз])

Шаг 4

Выделим цилиндр, внутри которого будет создана рабочая ось




Построение рабочей оси по трём (двум) взаимоперпендикулярным рабочим плоскостям

Чтобы построить рабочую ось:

Шаг 1

Введём AMBASICPLANES и нажмём клавишу <Enter>
Щёлкнем мышью на экране, чтобы разместить рабочие плоскости. (При этом создаются базовые рабочие плоскости XYZ).


Шаг 2

Введём с клавиатуры 8 и нажмём клавишу <Enter> (При этом на экране появится изображение трех базовых плоскостей в изометрической проекции).
Сделаем одну из плоскостей активной эскизной плоскостью.


Введём AMSKPLN и нажмём клавишу <Enter>
Щелкнем мышью на рабочей плоскости и дважды нажмём клавишу <Enter>.

Шаг 3

Зададим рабочую ось.
Для этого выполним команду AMWORKAXIS. Mechanical запросит:

Select cylinder, cone, torus or [Sketch]: (Выберите цилиндр, конус, тор или [эскиз])

Выбираем Sketch, что означает, что означает эскиз рабочей оси.
По двум точкам нарисуем на текущей активной эскизной плоскости линию. (Для размещения линии точно в центре рабочей плоскости воспользуемся осями привязки объекта или выберем опцию Ortho.)



Построение рабочей оси по двум точкам существующего отрезка

Предположим, имеется отрезок, который будет определять положение и направление построения рабочей оси:

Шаг 1

Щёлкнем правой кнопкой мышки по графическому полю.
На экране раксроется контекстно-зависимое меню.

Шаг 2

Щелчком по пункту Sketched&Work Features (Эскизные и рабочие элементы) в контекстно-зависимом меню вызовем всплывающее меню.

Шаг 3

Щёлкнув в нём по пункту Work Axis (Рабочая ось), увидим следующее сообщение:

Draw a two point line on the current sketch plane.

(Выберите цилиндр, конус, тор или [эскиз])

Шаг 4

Выберем две точки отрезка, по которым будет создана рабочая ось.
