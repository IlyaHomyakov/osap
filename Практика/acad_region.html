Командная строка

REGION

Области образуются объектами, определяющими замкнутые контуры. Контуры могут состоять из отрезков, полилиний, окружностей, дуг, эллипсов, эллиптических дуг и сплайнов. Контур должен быть замкнутым, т.е. либо состоять из одного замкнутого объекта, либо из замкнутой последовательности объектов, соединяющихся в конечных точках.

Для областей можно:

  • выполнять штрихование и закрашивание;
  • анализировать такие свойства, как площадь, момент инерции применяя команду MASSPROP (используется для, например, вычисления распределённых нагрузок, таких, как давление жидкости на плоскость, или же при расчёте сил, действующих в сгибаемой, или вращаемой балке);
  • получать информацию, например, о расположении центра масс, периметре.

Области могут быть созданы из нескольких контуров, а также из нескольких разомкнутых кривых, соединённых между собой и образующих контур. Нельзя формировать области из разомкнутых объектов, которые, хотя и образуют замкнутые фигуры, но пересекаются не в конечных точках: например, пересекающиеся дуги или самопересекающиеся кривые.

Команда REGION создает область из существующих объектов, которые образуют замкнутую границу.

Запрос:

Select objects:
Выберите объекты:

Запрос выдается циклически.

Можно создавать область из перекрывающихся объектов.

Меню

Home | Draw | Boundary

Командная строка

BOUNDARY

Команда BOUNDARY создает область из перекрывающихся объектов.

При этом используется диалоговое окно Boundary Creation (создание контура).


Для создания области необходимо:

  • в раскрывающемся списке Object Type (тип объекта) выбрать Region (область);
  • нажать кнопку Make New Boundary Set (создание нового контура);
  • выбрать объекты, участвующие в образовании области;
  • выбрать Pick Points (указание точек);
  • указать точку.

Составные области строятся путём вычитания, объединения и пересечения имеющихся областей. После этого к ним можно применять штрихование и подсчитывать общую площадь.
