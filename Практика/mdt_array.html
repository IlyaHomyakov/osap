Создаёт прямоугольные и полярные массивы элементов на активной детали.

Команда AMARRAY для создания прямоугольного или полярного массива элементов на активной детали. AMARRAY создает массивы эскизных элементов, элементов выдавливания, вращения, протягивания, отверстий и комбинированных элементов.

Кнопка инструментов
(Array)

      

Меню

Part -> Placed Features -> Array

Команда

AMARRAY

Диалоговое окно Array имеет две кнопки: Rectangular и Polar.


Rectangular

Columns along X axis

Определяет число столбцов массива и расстояние между ними.

Rows along Y axis

Определяет число рядов массива и расстояние между ними.

Для создания массива в отрицательном направлении осей X и Y, введите отрицательные значения в поле Spacing для рядов и столбцов.


Polar

Number of instances

Определяет число создаваемых элементов. Выбранный элемент считается первым элементом.

Angle

Определяет внутренний угол или угол между элементами массива, если задан не полный круг.

Введите положительное значение для создания массива в направлении против часовой стрелки или отрицательное значение для создания массива в направлении по часовой стрелке.

Angel type

Определяет тип используемого угла.

Full Circle. Создает массив на 360°.

Included Angle. Создает массив через указанный угол.

Incremental Angle. Создает массив с указанным приращением.

Rotate as copied

Определяет, поворачивать ли элемент по мере копирования. Для размножения элементов без поворота, Вы должны сначала создать рабочую точку в качестве ссылочной точки массива. Элементы выстраиваются вокруг центральной точки массива относительно этой рабочей точки.

Если Вы выбираете выстраивание выбранных элементов без поворота, отобразятся следующие подсказки:

Select work point, work axis or cylindrical surface for center of array:

Выберите рабочую точку,рабочую ось или цилиндрическую поверхность

Select work point to be used as reference point:

Выберите рабочую точку

OK

Выходит из диалогового окна Array и создает массив. Прямоугольный массив создается автоматически. Полярный массив выдает следующий запрос:

Select work point, work axis or cylindrical surface for center of array:

Выберите рабочую точку, рабочую ось или цилиндрическую поверхность

Выбор рабочей точки или рабочей оси. Если Вы выберите рабочую точку или рабочую ось, массив создается автоматически.

Выбор цилиндрической поверхности. Если Вы выберите цилиндрическую поверхность, сначала будет создана рабочая ось через центр плоскости и, затем, массив.

DONE

Выходит из диалогового окна Array и команды. Возвращается в диалоговое окно Array после того, как массив будет создан.
