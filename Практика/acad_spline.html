Создание сплайнов

Кнопка инструментов
Spline
     

Меню

Home | Draw | Spline

Командная строка

SPLINE

Сплайн - это гладкая кривая, проходящая через заданные точки. AutoCAD строит особый тип сплайна, известный как неоднородная рациональная В - сплайновая кривая - NURBS. NURBS кривые образуют гладкую кривую между контрольными точками. Сплайны обычно используются для представления кривых, имеющих нерегулярную форму, например, для рисования контурных линий в географических информационных системах или для дизайна автомобиля.

Команда SPLINE строит гладкую кривую по заданным определяющим точкам и направлению касательных в начальной и конечной точках.

Запрос:

Object/<Enter first point>:
Объект/<Введите первую точку>:

Object (объект) - преобразование 2D или 3D сглаженной полилинии в эквивалентный сплайн и удаление информации о полилинии. При вводе ключа О появится запрос на выбор объекта.

После ввода первой точки появятся запросы:

Enter point:
Введите точку:
Close/Fit Tolerance/<Enter point>:
Закрыть/Допуск/<Введите точку>:

Fit Tolerance (допуск) - при нулевом значении сплайн проходит через заданные точки, при положительном - с заданным отклонением.

Для определения касательных в начальной и конечной точках незамкнутой кривой выдаются соответственно запросы:

Enter start tangent:
Направление касательной начала:
Enter end tangent:
Направление касательной конца:

В замкнутой кривой (ключ Close) для ввода направления касательной выдается запрос:

Enter tangent:
Направление касательной.


Редактирование сплайнов

Кнопка инструментов
Edit Spline
     

Меню

Home | Modify | Edit Spline

Командная строка

SPLINEDIT

Выдается запрос:

Select spline:
Выберите сплайн:

Далее следует запрос:

Close/Move Vertex/Refine/rEverse/Undo/eXit <X>:

Выберите ключ или нажмите ENTER

Если выбранный сплайн замкнут, ключ Close (Замкнуть) заменяется на Open (Разомкнуть).

Move Vertex (Переместить вершину) - перемещает контрольные узловые точки сплайна и очищает точки соответствия.

Refine (Уточнить) - точно регулирует описание сплайна.

rEverse (реверс) - меняет направление сплайна. Этот ключ, в основном, для использования приложениями.

Undo (Отменить) - отменяет последнюю операцию редактирования.

eXit (выход) - заканчивает выполнение команды SPLINEDIT.
