Штриховка – это обычные отрезки Автокада, отрисованные, в соответствии с шаблоном и объединённые во внутренний блок. Отсюда следует несколько свойств штриховки:

  • штриховка – это блок состоящий из нескольких объектов, поэтому для удаления штриховки достаточно указать на любую из её линий;
  • при отрисовке штриховки действительны все текущие общие свойства примитивов. Поэтому перед выполнением штриховки рекомендуется устанавливать непрерывный тип линии;
  • штриховка всегда создается в текущей ПСК, даже если указанный контур располагается вне плоскости ПСК.

Создание штриховки

Кнопка инструментов
Hatch
(панель Draw)
     

Меню

Home | Draw | Hatch...

Команда

HATCH

Командная строка

BHATCH

По команде ВНАТСН на экран выводится диалоговое окно штриховки по контуру Boundary Hatch. Все значения по умолчанию в этом окне берутся из значений, определенных при последнем вызове команды ВНАТСН или HATCH.


Меню штриховки


Образец штриховки (Pattern Type)

Образец штриховки состоит из одной или нескольких линий штриховки, которые могут быть непрерывными или штрихпунктирными. Чтобы заполнить область штриховки, образец повторяется или подрезается. Так как образец может включать много линий, AutoCAD объединяет их во внутренний блок.

При нажатии на кнопку Образец (Pattern), на экран выведется диалоговое окно образцов штриховки Hatch pattern palette, которое позволяет выбрать существующий образец штриховки. Для выбора образца штриховки следует указать его изображение или выбрать название из списка.

Для использования стандартных образцов штриховки необходимо в окне Boundary hatch в выпадающем списке Type выбрать Стандартный (Predefined).

При выборе в списке Пользовательский (User-defined) штриховка выполняется с использованием текущего типа линии.

Обычный (Custom) устанавливает образец из файла acad.pat или любого другого файла с расширением .pat.


Свойства штриховки (Pattern Properties)

Ширина пера ISO (ISO Pen Width). Масштабируется как относящийся к ISO стандарту образец, базирующийся на выбранной ширине пера. Эта опция доступна только при выбранном типе Predefined и одной из штриховок ISO.

Образец (Pattern). Образец штриховки можно выбрать по его названию из выпадающего списка Pattern, причем вид данного образца автоматически отображается в области Pattern Type.

Обычный образец (Custom Pattern). Устанавливает имя образца. AutoCAD хранит имя образца в системной переменной HPNAME. Эта опция доступна только при выбранном типе Custom.

Масштаб штриховки (Scale). Опция Масштаб штриховки предоставляет вам возможность изменять плотность штриховки выбранного образца, обеспечивая тем самым лучшее зрительное различение смежных изделий, имеющих по каким - либо причинам одинаковый образец штриховки.

При единичном масштабе штриховки (для самого распространенного образца ANSI31) расстояние между соседними линиями равно 0.125 усл. ед. (такая величина в Автокаде изначально связана с дюймовой системой счисления). Поэтому для ориентации использования масштаба штриховки в чертежах рекомендуется ввести поправочный коэффициент, равный восьми, иными словами, в штриховке с именем образца ANSI31 и с масштабом 8 расстояния между соседними линиями равняются 1 усл. ед.

Угол штриховки (Angle). При использовании типового образца штриховки угол 0° обозначает, что наклон штриховки в чертеже от оси Х текущей ПСК будет таким же, как на образце; поэтому если в качестве образца вы используете штриховку ANSI 31 и хотите штриховать с таким же углом наклона, как на образце, то задаваемый угол должен равняться 0°.

Расстояние (Spacing). Расстояние между линиями штриховки. Запоминается в системной переменной HPSPACE. Эта опция доступна только при выбранном типе User-defined.

Крест-накрест (Double). Переключатель Double используется для отрисовки дополнительных линий под углом 90° к основным линиям штриховки (при этом системная переменная HPDOUBLE устанавливается в 1). Эта опция доступна только при выбранном типе User-defined.


Контур (Boundary)

Выбор точек (Pick Points<). Создает контур из существующих объектов, которые представляют собой очаг. Выберите эту опцию для подтверждения установок и создания контура.

Select internal point:
Выберите внутреннюю точку.

Если вы выбрали точку в очаге, AutoCAD подсветит контур. AutoCAD будет продолжать запрашивать внутреннюю точку до тех пор, пока вы не нажмете ENTER.

Если вы выбрали точку вне очага, AutoCAD отобразит диалоговое окно Definition Error (Ошибка выбора).

Выбор объектов (Select Objects<). Выбирает определенные объекты для штрихования. Диалоговое окно Boundary Hatch закрывается и AutoCAD просит выбрать объект.

Когда вы используете опцию Выбор объектов, AutoCAD не обнаруживает внутренних объектов автоматически. Вам необходимо выбрать все внутренние объекты, которые не нужно заштриховывать.

Каждый раз, когда вы используете опцию Выбор объектов, AutoCAD отменяет предыдущий выбор.

Удаление изолированных участков (Remove Islands<). Удаляет из контура объекты, определенные как изолированные участки опцией Выбор точек. Нельзя удалить внешний контур.

Отображение выбора (View Selections<). Отображает контур, выбранный в настоящее время. Эта опция недоступна, когда ничего не выбрано или контур создан.

Дополнительные возможносит (Advanced...). Показывает диалоговое окно Advanced Options.

Предварительный просмотр штриховки (Preview Hatch<). Показывает, какой будет штриховка. AutoCAD закрывает диалоговое окно и штрихует выбранную область. После просмотра штриховки выберите Продолжить (Continue) для возвращения в диалоговое окно. Эта опция не доступна, если ничего не было выбрано.

Наследуемые свойства (Inherit Properties<). Принимает свойства существующей штриховки. Диалоговое окно закрывается и AutoCAD просит выбрать штриховку, которая будет образцом. После выбора заштрихованного объекта вновь появляется диалоговое окно Boundary Hatch, в котором отображаются свойства выбранной штриховки.


Атрибуты (Attributes)

Соединяющий (Associative). Контроль соединения штриховки. Если эта опция выбрана, в новой штриховке все линии объединены во внутренний блок.

Разъединяющий (Exploded). Обозначает, что штриховка была создана из индивидуальных линий. AutoCAD хранит эту информацию, помещая звездочку перед именем штриховки в системной переменной HPNAME. Вы также можете использовать EXPLODE (Разбиение) для разбиения цельной штриховки на отдельные линии.


Определение области штриховки (Boundary)

Выбор определяющих контур объектов производится так же, как в других командах: можно выбирать объекты рамкой, прицелом, удалять объекты из набора, пополнять набор и т. д. После завершения выбора объектов Автокад выполняет штриховку выбранного образца. Т.к. Автокад не анализирует пересекающиеся примитивы на предмет того, образуют ли они замкнутую область, то контур штрихования должен быть замкнут и состоять из примитивов (отрезки, дуги, окружности, полилинии или грани), имеющих общие конечные точки. При построении чертежа лучше всего заранее учитывать эту особенность штриховки и отрисовывать будущие контуры штрихования одной командой Line (Отрезок) или Pline (ПЛиния) с использованием опции Close (Замкни). При штриховке контура, ограниченного полилинией ненулевой ширины или полосой, контур штриховки определяется осевой линией этих примитивов. Штриховка создается в плоскости текущей ПСК, и если вы пожелаете заштриховать переднюю грань пирамиды, а текущая система координат при этом осталась в горизонтальной плоскости, то вы получите штриховку горизонтальной проекции упомянутой грани.

Когда мы заштриховываем несколько отдельных объектов с одинаковым масштабом и углом наклона, то Автокад формирует узор таким образом, что, если бы штриховка продолжилась за границы одного объекта, она точно совпала бы со штриховкой другого.


Штрихование из командной строки

Команда HATCH обеспечивает штрихование из командной строки.

Первоначально выдается запрос:

Enter pattern name or [? /Solid/User defined]<current>:
Введите имя образца или [? /Заливка/Cоздание образца]<по умолчанию>:

Он содержит следующие ключи:

Имя (Name). Позволяет задать образец штриховки, описанный заранее. Обычно каждая команда HATCH формирует блок, содержащий все штриховые линии. Чтобы отказаться от этого, необходимо перед именем задать звездочку (*). По умолчанию стоит имя последней используемой штриховки.
После задания имени образца выдаются два дополнительных запроса:

Scale for pattern <current>:
Масштаб штриховки <по умолчанию>:

Angle for pattern <current>:
Наклон штриховки <по умолчанию>:

? Служит для вывода списка образцов штриховки.

Заливка (Solid). Заполнение подобно заполнению фигуры - заливка цветом.

Cоздание образца (User defined). Служит для непосредственного определения вида штриховки.
Дополнительные запросы:

Angle for crosshatch lines <current>:
Угол наклона штриховки <по умолчанию>:

Spacing between lines <current>:
Шаг штриховки <по умолчанию>:

Double hatch area? <no>:
Штриховать крecт-накрест? <нет>:

После того как задан тип штриховки, AutoCAD выдает запрос:

Select objects:
Выберите объекты для выбора контура штриховки.


Стили штрихования

При выполнении штриховки можно выбирать не только ее образец, но и стиль (Boundary Style в окне Advanced Options ).


Расширенные опции

В Автокаде предусмотрено 3 стиля штриховок: Нормальный (Normal), Внешний (Outer) и Игнорирующий (Ignoring). Следует отметить, что стиль действует только в том случае, когда внутри контура штрихования содержатся какие-то объекты. Если пространство внутри области штриховки пусто, то оно заполняется выбранным образцом. Ниже на рисунке показан пример использования стилей штрихования.

Нормальный стиль (Normal) штриховки устанавливается по умолчанию (рисунок слева). Штрихование начинается с внешнего контура и при нахождении внутреннего контура штрихование не прекращается до тех пор, пока не будет найден еще один вложенный контур. Таким образом, все нечетные области будут заштрихованы, а четные нет. Вокруг текста создается невидимый прямоугольник, прерывающий штриховку.

Внешний стиль (Outer) штриховки закрашивает только внешний контур (рисунок справа). При нахождении вложенного контура штрихование прекращается (ни один вложенный контур не будет заштрихован).

Игнорирующий стиль (Ignoring) отключает режим поиска и анализа вложенных контуров штриховки (рисунок внизу). Автокад заштриховывает всю область, определяемую внешним контуром, игнорируя любые внутренние контуры.

Образец, стиль, поворот и масштаб штриховки, которые использовались в последней команде HATCH, становятся значениями по умолчанию при последующих вызовах команды HATCH или ВНАТСН в течение текущего сеанса работы AutoCAD. Кроме того, если повторить команду HATCH сразу же (нажав или <Пробел> в ответ на первую же подсказку Command:), AutoCAD считает, что требуется заштриховать другую область, используя те же образец, стиль, масштаб и угол поворота. При этом AutoCAD опускает соответствующие запросы и сразу выводит подсказку Select objects: (Выберите объекты).


Редактирование штриховки

Кнопка инструментов
Edit Hatch
(панель Modify II)
     

Меню

Modify | Edit Hatch

Командная строка

HATCHEDIT

После выбора редактирования штриховки появится запрос:

Select hatch object:
Выберите объект штриховки:

После выбора штриховки, которую необходимо редактировать, появится диалоговое окно редактирования штриховки Hatchedit. По своему составу оно полностью совпадает с описанным выше окном штриховки по контуру Boundary Hatch.
