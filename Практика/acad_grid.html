      Изометрические шаг и сетка (Isometric Snap/Grid)

Координатная сетка служит для лучшей ориентации пользователя при черчении. Она не влияет на сам чертеж и не отображается при распечатке.

Включить или выключить отображение координатной сетки можно с помощью клавиши F7 либо с помощью кнопки GRID в статусной строке.

Настроить сетку можно с помощью команды GRID. При ее выполнении появляется запрос:

Здесь предлагается ввести шаг сетки, значение по умолчанию находится в угловых скобках. Если ввести ON, сетка будет включена, OFF - выключена. При вводе Snap, шаг сетки будет приравнен к шагу перемещения курсора мыши. При вводе Aspect будет дан запрос отдельно на горизонтальный и вертикальный шаг сетки:

Horizontal spacing(X) <current>:
Vertical spacing(X) <current>:

Во многих случаях при черчении удобнее, когда курсор мыши передвигается не плавно, а некоторыми шагами, например, в 1 мм. Для этого в AutoCAD существует так называемая привязка SNAP. Включение/отключение может осуществляться с помощью клавиши F9 либо с помощью кнопки Snap в статусной строке программы. Шаг перемещения мыши можно задавать с помощью команды SNAP.

При ее выполнении появляется запрос:

Пользователю предлагается ввести шаг перемещения мыши, значение по умолчанию задается в угловых скобках. Если ввести ON, привязка будет включена, OFF - выключена. При вводе Aspect будет дан запрос отдельно на горизонтальный и вертикальный шаг мыши:

Horizontal spacing <current>:
Vertical spacing <current>:

Опция Rotate позволяет сделать привязку не вдоль осей координат, а повернутой под некоторым уголом:

Base point <current>:
Rotation angle <current>:

Сначала пользователя просят ввести координаты точки, относительно которой будет осуществляться поворот, а во втором запросе пользователь должен ввести угол, на который поворачивается сетка привязки.

Опция Style позволяет переключаться между стандартным и изометрическим форматом привязки.

Опция Type позволяет переключаться между привязкой к координатной сетке и привязкой к полярным координатам.


Диалоговое окно Режимы рисования (Drawing Aids)

Настройка координатной сетки и привязки мыши может осуществляться с помощью диалогового окна Drawing Aids.

Меню

Tools | Drafting Settings

Командная строка

DSETTINGS


Режимы (Modes)

  • Орто (Ortho) При включении данного режима черчение становится возможным только строго вертикально или горизонтально относительно предыдущей введенной точки. Данный режим также включается/выключается с помощью клавиши F8 либо с помощью кнопки ORTHO, находящейся в статусной строке.
  • Закраска (Solid Fill) - включает/выключает режим закрашивания, применяемый для некоторых объектов (фигур, полилиний и их производных);
  • Конт. текст (Quick Text) - режим отображения текста и атрибутов в виде простых прямоугольников;
  • Маркеры (Blips) - включает/выключает временные графические крестообразные значки, размещаемые системой в заданной вами точке;
  • Подсветка (Highlight) - включает/выключает подсветку выбранных объектов;
  • Группы (Groups) - включает/выключает автоматическое выделение групп;
  • Штриховка (Hatch) - определяет, какие объекты будут выбраны при выборе ассоциативной штриховки.

Шаг (Snap)

  • Вкл (On) - включает/выключает привязку мыши;
  • Интервал по X (X Spacing) - поле, в котором устанавливается шаг мыши по оси X;
  • Интервал по Y (Y Spacing) - поле, в котором устанавливается шаг мыши по оси Y (в случае, если значения не вносились автоматически, становится равным значению предыдущего поля);
  • Поворот сетки (Snap Angle) - уход поворота привязки курсора;
  • База по X (X Base) - координата Х базовой точки;
  • База по Y (Y Base) - координата Y базовой точки.

Сетка (Grid)

  • Вкл (On) - включает/выключает координатную сетку;
  • Интервал по X (X Spacing) - шаг координатной сетки по оси X;
  • Интервал по Y (Y Spacing) - шаг координатной сетки по оси Y (в случае, если значения не вносились, автоматически становится равным значению предыдущего поля).

Изометрические шаг и сетка (Isometric Snap/Grid)

  • Вкл (On) - включает/выключает изометрический стиль сетки;
  • Левая (Left) - включает левую плоскость изометрии;
  • Верхняя (Top) - включает верхнюю плоскость изометрии;
  • Правая (Right) - включает правую плоскость изометрии.
