Угловая связь деталей узла

    Угол между двумя векторами
    Угол между двумя плоскостями
    Угол между плоскостью и вектором

Опция Angle используется для размещения поверхностей под углом друг к другу (заставляет поверхности или ребра располагаться под углом).

С помощью команды AMANGLE создаётся угловая связь деталей узла, подузлов и узлов. Эта команда позволяет управлять углом между двумя плоскостями, двумя векторами или комбинацией плоскости и вектора. Плоскости и векторы могут быть определены комбинацией точек, осей и плоскостей.

Точка задается конечной точкой линии, конечной гранью тела, центром дуги, сферы или тора, рабочей точкой, точкой AutoCAD, точкой на выделенной поверхности грани и определяет ближайшую точку привязки.
Ось устанавливается линией или линейной кромкой тела, рабочей осью, двумя точечными примитивами, плоскостью.

Плоскость указывается плоской гранью детали, дугой, рабочей плоскостью, двумя осевыми примитивами, точкой и осью, тремя точечными примитивами.

Обращение к команде AMANGLE может быть выполнено несколькими способами:

  • щелкните правой кнопкой мыши в графической области, чтобы на экране развернулось контекстно-зависимое меню. Щелкните по пункту 3D Constraints Angle (Трехмерные угловые связи); щелкните по кнопке Angle (Угол) на панели инструментов;
  • щелкните по пункту Assembly (Узел) главного меню, по пункту 3D Constraints (Трехмерные связи) выпадающего меню, а затем по пункту Angle всплывающего меню;
  • введите в командной строке команду AMANGLE.

    Рассмотрим различные варианты управления углом.


    Угол между двумя векторами

    В этом случае появится запрос:

    Select first set of geometry:
    (Определите первый набор геометрии)
    First set = Axis/Vector, (line)
    (Первый набор = Ось/Вектор (линия))
    Select first set or [Clear/fAce/Point/Flip/cYcle]<accept>:
    (Выберите первый набор или [Очистить/Грань/Точка/Отразить зеркально/ Цикл]<принять>)

    Выполните следующие действия:

    1. Задайте первый набор или опцию либо нажмите клавишу Enter. Появится запрос:

    Select second set of geometry:
    (Выберите второй набор геометрии)
    Second set = Axis/Vector, (line)
    (Второй набор = Ось/Вектор (линия))
    Select second set or [Clear/fAce/Point/Flip/cYcle]<accept>:
    (Выберите второй набор или [Очистить/Грань/Точка/Отразить зеркально/ Цикл]<принять>)

    2. Установите второй набор или опцию либо нажмите на Enter. Появится запрос:

    Angle <0>:
    (Угол <0>)

    3. Определите угол или нажмите клавишу Enter.


    Угол между двумя плоскостями

    В этом случае появится запрос:

    Select first set of geometry:
    First set = Plane/Vector
    (Первый набор = Плоскость/Вектор)
    Enter an option [Clear/Next/Flip/cYcle]<accept>:
    (Введите опцию [Очистить/Следующая/Отразить зеркально/Цикл]<принять>)

    Выполните следующие действия:

    1. Укажите опцию или нажмите на Enter. Повторится такой запрос:

    Select second set of geometry:

    2. Определите другую плоскость, выбрав геометрию. Последует запрос:

    Second set = Plane/Vector
    (Второй набор = Плоскость/Вектор)
    Enter an option [Clear/Next/Flip/cYcle]<accept>:

    3. Отметьте опцию или нажмите клавишу Enter. Поступит предложение:

    Enter angle <0>:

    4. Определите угол или нажмите на Enter.


    Угол между плоскостью и вектором

    Запрос выглядит так:

    Select first set of geometry:
    First set = Plane/Vector
    Enter an option [Clear/Next/Flip/cYcle]<accept>:

    Выполните следующие действия:

    1. Выделите опцию или нажмите на Enter. Поступит запрос:

    Select second set of geometry:
    Second set = Axis/Vector, (line)
    Select second set or [Clear/fAce/Point/Flip/cYcle]<accept>:

    2. Укажите опцию или нажмите на Enter. Появится известный запрос:

    Angle <0>:

    3. Определите угол и нажмите клавишу Enter.
