Home | Modify | Offset

Командная строка

OFFSET

Команда OFFSET используется для создания подобных объектов (эквидистанты) с заданным смещением.

Запросы команды Offset:

Offset distance or Through <1.0000>:
Величина смещения или Точка <1.0000>:

Указать смещение, то есть число или пару точек.

Select object to offset:
Выберите объект для создания ему подобных:
Specify point on side to offset:
Укажите сторону, в которую необходимо смещать создаваемый объект:
Select object to offset:

Нажмите Enter для завершения работы команды.

Ключ Through (Точка) позволяет задавать смещение через точку. При этом выдаются следующие запросы:

Offset distance or Through <default>: t

Перейти в режим указания смещения через точку.

Select object to offset:
Specify through point:
Укажите точку:
Select object to offset:

Нажмите Enter для завершения работы команды.

Select object to offset: - указать полилинию
Side to offset? - указать точку ниже полилинии

До редактирования

После

Слои подобны лежащим друг на друге прозрачным листам кальки. На различных слоях группируются различные типы данных рисунка. Использование цвета позволяет различать сходные элементы рисунка. Применение линий различных типов помогает быстро распознавать такие элементы, как осевые или скрытые линии. Расположение объектов на различных слоях позволяет упростить многие операции по управлению данными рисунка.
