Привязка к объектам

Достаточно часто некоторую точку рисуемого объекта следует привязать к точке, расположенной на ранее начерченном объекте. Для этого служит режим объектной привязки. Включается/выключается данный режим клавишей F3 либо кнопкой OSNAP в статусной строке программы. Настройка объектной привязки осуществляется с помощью диалогового окна Osnap Settings.

Кнопка инструментов
Object Snap Settings
     

Командная строка

OSNAP

В диалоговом окне присутствуют следующие опции:

Endpoint (Конечная точка) - привязка к конечной точке объекта.

Midpoint (Середина) - привязка к середине объекта.

Center (Центр) - привязка к центру окружности либо дуги.

Node (Узел) - привязка к точечному объекту.

Quadrant (Квадрант) - привязка к квадрату на окружности, дуге, эллипсе и т.д.

Intersection (Пересечение) - привязка к точке пересечения двух объектов.

Insertion (Точка вставки) - привязка к точке вставки блоков и подобных объектов.

Perpendicular (Нормаль) - привязка к точке, которая вместе с предыдущей точкой образует перпендикуляр к данному объекту.

Tangent (Касательная) - привязка к точке, которая вместе с предыдущей точкой образует касательную к окружности, дуге, эллипсу и т.п.

Nearest (Ближайшая) - привязка к ближайшей точке на объекте.

Apparent Int (Кажущееся пересечение) - привязка к точке видимого пересечения двух объектов, которые не пересекаются в трехмерном пространстве, но имеют видимую точку пересечения в текущем виде.

Quick (Быстрая) - вспомогательный режим, служит для ускорения работы остальных операций захвата за счет быстрого нахождения первой же точки, удовлетворяющей условию захвата.

Clear All (Отменить все) - отключение всех видов привязок.
Select All - включение всех видов привязок.
