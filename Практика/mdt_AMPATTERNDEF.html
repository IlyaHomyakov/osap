Mechanical Desktop














Штриховка


Для определение образца штриховки в Mechanical Desktop существует команда AMPATTERNDEF

Кнопка инструментов

      Hatch Pattern

Меню

Assembly -> Assembly -> Hatch Patterns

Команда

AMPATTERNDEF


Создание штриховки

После этого откроется диалоговое окно Hatch Pattern. В разделе Parts/Subassemlies перечисляются определения детали и подузлы в текущем чертеже. Раздел Patter Type позволяет установить предопределённые, задаваемые пользователем или заказные образцы штриховки. В разделе Pattern Properties назначаются свойства, которые будут применены к выбранному образцу штриховки.

Iso Pen Width - служит для указания ширины пера, применяемой к образцу штриховки;

Pattern - определяет предопределённый образец штриховки;

Custom Pattern - предназначен для ввод заказного образца штриховки;

Scale - устанавливает размер образца штриховки;

Angle - угол образца штриховки;

Spacing - интервал образца штриховки;

Double - двойное изображение образца штриховки;

Exploded - вырезанное изображение образца штриховки.

1. Чтобы применить штриховку для определённой детали, необходимо знать её оригинальное имя. По нажатию правой кнопкой мыши на деталь – Rename definition откроется диалоговое окно Rename Definition.

2. В окне написано название детали (Old name), а так же имеется возможность изменить его, для более удобного нахождения детали в окне нанесения штриховки.

3. После закрытия окна необходимо вызвать команду AMPATTERNDEF. В диалоговом окне выбрать из списка деталь, для которой нужно определить штриховку. Настроить штриховку с нужными параметрами.

Таким способом необходимо определить штриховку для всех деталей.

Далее необходимо перейти в Drawing Mode

Необходимо выбрать вид на котором есть штриховка, далее перейти в его настройки. Во вкладке Section по нажатию кнопки Pattern откроется диалоговое окно Hatch Pattern.

В окне в заголовке Pattern Type, можно установить предопределённые, задаваемые пользователем или заказные образцы штриховки.

Predefined (Предопределённый) - для всех деталей определяет один образец штрихвоки.

User-defined (Задаваемые пользователем) - для каждой отдельной детали определяет образец штриховки. Его и необходимо выбрать.

Далее необходимо применить изменения и обновить вид. Каждая деталь будет заштрихована по нужному образцу.


Цвет штриховки

Для изменения цвета штриховки, необходимо открыть Model Mode.

По нажатию правой кнопкой мыши по детали, далее Properties – Color. И устанавливаем цвет. Так необходимо проделать для каждой детали, для которой необходимо изменить цвет штриховки.

Для того чтобы у каждой детали штриховка была определённого цвета необходимо открыть в главном меню Drawing – Drawing Options.

Далее найти пункт Hatch и установить в нём значение Use Part Color.

В Drawing Mode обновить виды.
