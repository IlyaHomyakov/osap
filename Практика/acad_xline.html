Командная строка

XLINE

Команда XLINE строит бесконечные линии, которые обычно используютя в качестве линий построения.

Запрос:

Hor/Ver/Ang/Bisect/Offset/<<From point>:

Ключи:

Hor - построение горизонтальной прямой, проходящей через заданную точку.
Ver - построение вертикальной прямой, проходящей через заданную точку
Ang - построение прямой по точке и углу.
Bisect - по точке и половине угла, заданного тремя точками.
Offset - по смещению от базовой линии.

По умолчанию строятся прямые по двум точкам.
