Включение и настройка ручек

Выбрав какой-либо объект, можно изменять его вид с помощью ручек (grips) - маленьких квадратов, которые высвечиваются в определенных местах объекта.

Меню

Tools | Grips

Командная строка

DDGRIPS

Настройка ручек осуществляется с помощью диалогового окна Grips.


Выбор установок (Select settings)

Enable Grips (Включение ручек) - включает/выключает использование ручек.
Enable Grips within Block (Ручки внутри блоков) - когда включен данный пункт, при выборе блока появляются ручки всех объектов, в него входящих. В противном случае появляется одна ручка в точке вставки блока.


Цвета ручек (Grip colors)

Unselected (Невыбранные) - в данном выпадающем списке можно выбрать цвет для ручек, не выбранных в данный момент.
Selected (Выбранные) - в этом выпадающем списке можно выбрать цвет ручек, выбранных в данный момент.


Размер ручек (Grip Size)

C использованием данного пункта можно изменять размер квадрата, обозначающего ручку.


Процесс редактирования

Для того, чтобы начать редактирование с помощью ручек, следует выбрать одну из них, после чего происходит переход в режимы редактирования с помощью ручек. Всего их пять: Stretch (растягивание), Move (перенос), Rotate (поворот), Scale (масштабирование), Mirror (зеркало). По умолчанию после начала редактирования ручками включается режим растягивания. Для переключения между режимами существует несколько способов:

  • Режимы переключаются циклически один за одним при нажатии пробела.
  • Выбрать режим можно, набрав его название в командной строке.
  • Выбрать режим можно из контекстного меню, вызываемого правой кнопкой мыши.

Кроме опций выбора режима редактирования, следует также упомянуть еще две: Copy (копирование) и Base Point (базовая точка). Когда включен режим Copy, при редактировании элемента создается новый, но старый не удаляется, того же эффекта можно добиться, удерживая при редактировании клавишу SHIFT. Если выбрать Base Point, то будет запрошена точка, которая станет новой базовой точкой редактирования вместо той, где находится выбранная ранее ручка.

Режим растягивания работает аналогично команде STRETCH с некоторыми нюансами. Если команда STRETCH работает с конечными точками отрезков, дуг и центральными точками окружностей, то при работе с ручками можно использовать средние точки отрезков и квадраты окружностей. При использовании средних точек отрезков и центров окружностей объект перемещается без растягивания, то есть выполняется операция, аналогичная команде MOVE. В режиме растягивания можно работать сразу с несколькими примитивами, для чего при включении режима редактирования следует выбирать несколько ручек, удерживая нажатой клавишу SHIFT. Для начала редактирования следует указать последнюю ручку, отпустив SHIFT.

Работа остальных режимов практически полностью аналогична работе соответствующих команд: MOVE, MIRROR, ROTATE, SCALE.
